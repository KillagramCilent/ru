# Killagram Desktop readiness (Windows)

## Что уже подготовлено

1. Добавлен адаптивный вход в desktop-layout через ширину экрана (`>=1000`).
2. Добавлен 2-pane экран:
   - слева список чатов
   - справа активный чат
3. Мобильный путь сохранён: на узких экранах используется старый flow (список -> переход в чат).

## Desktop UX (как Telegram Desktop)

### Keyboard-first

- `↑ / ↓` — перемещение выбора по чатам (когда в фокусе список чатов).
- `Enter` — открыть выбранный чат.
- `Esc` — вернуть фокус в список чатов.
- `Ctrl + F` — перевести фокус в поиск.

### Mouse-first

- Hover-подсветка строки чата.
- Курсор `click` над чатами.
- Увеличенные hit-area/отступы на desktop.
- Отдельная подсветка выбранного чата.

### Context menu

- Right click по чату:
  - `Mute`
  - `Mark as read`
  - `Delete` (stub)
- Right click по сообщению:
  - `Copy text`
  - `Reply` (stub)

### Focus model

- Левая панель (список чатов) и правая панель (активный чат) имеют явный фокус.
- Фокус подсвечивается рамкой.

## Почему это важно

- На Windows широкие экраны требуют одновременного показа списка и чата.
- Такой UX совпадает с Telegram Desktop и ускоряет работу пользователя.

## Пошагово для локального запуска Windows

1. Установить Flutter SDK с поддержкой desktop.
2. Включить Windows desktop:
   ```bash
   flutter config --enable-windows-desktop
   ```
3. Проверить окружение:
   ```bash
   flutter doctor -v
   ```
4. Из директории `killagram_mobile` подтянуть пакеты:
   ```bash
   flutter pub get
   ```
5. Запустить на Windows:
   ```bash
   flutter run -d windows
   ```
